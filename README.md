# Movie Correlation Analysis

## Project Overview
This project analyzes correlations between different variables in a movie dataset to determine what factors influence box office revenue.

## Objectives
- Clean and preprocess movie data
- Handle missing values
- Convert data types
- Identify correlations between variables
- Visualize relationships using plots

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Key Analysis
- Scatter plot of budget vs gross earnings
- Correlation matrix
- Heatmap visualization

## Key Findings
- Strong correlation between **budget and gross revenue**
- Votes also show a significant correlation with gross earnings
- Score and other variables have weaker relationships

## Dataset
Movies dataset containing information about:
- Budget
- Gross earnings
- Votes
- Score
- Company
- Genre

## Conclusion
Budget and audience engagement (votes) are the strongest predictors of movie revenue.
